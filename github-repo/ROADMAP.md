# AIR Public Roadmap 2026-2027

**Last Updated**: 2026-03-18  
**Next Review**: 2026-06-18

---

## Overview

This roadmap outlines the Agent Identity Registry Foundation's priorities and initiatives for the next 18 months. We commit to:

- **Transparency**: Sharing our plans publicly
- **Flexibility**: Adjusting based on community feedback and technical learnings
- **Accountability**: Delivering on committed milestones
- **Community Input**: Incorporating stakeholder feedback quarterly

## Roadmap Principles

1. **Mission-Driven**: All initiatives advance neutral AI agent identity and trust
2. **Technical Excellence**: Solutions are technically sound and maintainable
3. **Community-Focused**: Prioritized based on community needs and feedback
4. **Security-First**: Security considerations integrated into all work
5. **Open Standards**: Build on W3C and other open standards

### Priority: Foundation Stabilization & Registry Launch

#### Core Initiatives

**1. Production Registry Launch** [IN PROGRESS]
- Complete: Core registry implementation (PostgreSQL backend)
- Complete: API v1.0 finalization
- Status: 95% complete; launch planned for April 15
- Deliverables: Public registry at agentidentityregistry.org with 100+ initial agents

**2. Trust Score v1.0 Finalization**
- Complete: Methodology specification
- Status: Community review period active (ends April 30)
- Deliverables: Published trust score algorithm; publicly auditable calculations

**3. Verifier Certification Program**
- Status: First cohort of 8 verifiers identified and training underway
- Deliverables: Verifier qualification standards; certified verifier directory

**4. Documentation & Developer Relations**
- Status: API documentation 80% complete
- Deliverables: Complete API docs; code examples in 3 languages; quickstart guides

**5. Community & Marketing**
- Status: Website and blog live
- Deliverables: Quarterly newsletter; social media presence; first community survey

#### Success Metrics

- 500+ community members engaged

#### Known Risks

- Regulatory uncertainties may require legal review

## Key Dependencies & Risks

### Critical Path Items

- Regulatory clarity on AI agent governance (external)
- Major AI platform adoption decisions (external)
- Funding for team expansion (controllable)
- Security vulnerabilities (manageable)

### Mitigation Strategies

**Regulatory Uncertainty**: Active engagement with policymakers; flexible specification

**Platform Adoption**: Strong technical design; clear value proposition; proactive outreach

**Funding**: Diversified revenue model; conservative budgeting; cost efficiency

**Security**: Regular audits; bug bounty program; rapid response protocols
